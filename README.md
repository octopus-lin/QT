项目背景（S）
针对Qt即时通讯应用开发需求，基于Qt 5.14.2框架开发的微信客户端与服务器系统。项目旨在实现类似微信的即时通讯功能，包括文本聊天、好友管理、视频通话等核心功能，同时提供友好的用户界面和稳定的通信体验。

职责与目标（T）
作为独立开发者，负责客户端和服务器端的完整开发工作。使用Qt框架的网络模块、数据库模块和UI组件，实现客户端与服务器的通信协议、用户认证、好友系统、聊天功能和视频通话功能。目标是构建功能完整、性能稳定、交互友好的即时通讯系统，支持多用户并发通信。

行动与技术亮点（A）
系统架构设计 ：采用客户端-服务器（C/S）架构，客户端使用Qt Widgets构建界面，服务器使用QTcpServer处理客户端连接
网络通信系统 ：
基于QTcpSocket实现可靠的文本数据传输，支持用户登录、注册、好友请求等功能
使用QUdpSocket实现视频通话功能，支持实时视频流传输
设计自定义通信协议，实现消息的封装与解析
数据库管理 ：
使用SQLite数据库存储用户信息、好友关系和聊天记录
实现用户认证、好友管理、消息存储等功能
客户端界面实现 ：
基于QMainWindow构建主界面，支持窗口拖动、最小化、关闭等操作
实现登录界面、聊天界面、好友列表、设置界面等多个功能界面
使用QMovie和QPainter实现界面动画效果
核心功能模块 ：
用户系统：支持注册、登录、修改个人信息、上传头像
好友系统：支持添加好友、查看好友列表、处理好友请求
聊天功能：支持文本消息、表情包、本地图片、截图发送与接收、聊天记录存储
视频通话：支持发起视频请求，实时视频

成果（R）
功能实现 ：完整实现即时通讯系统的核心功能，包括用户认证、好友管理、文本聊天、视频通话等
性能表现 ：系统运行稳定，支持多用户并发通信，视频通话流畅
技术收获 ：深入掌握Qt框架的网络编程、数据库操作、UI设计等核心技术，理解即时通讯系统的设计原理
